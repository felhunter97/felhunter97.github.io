hexo博客编写之markdown用法的学习

发表于 2018-03-26 | 分类于 学习
字数统计: 574

hexo博客写作

新建Markdown文件

cd到hexo博客的目录下，执行

hexo new “postName”

之后可以在~/blog/source/_posts目录下找到新建的.md文件，点击打开进行编写。编写完毕执行即可发布。

hexo clean
hexo g -d

修改文章标题以及添加分类和标签

在新建的.md文件打开之后最上面会有一段可编辑段落
title： 文章标题名称
date： 时间
categories： 分类
- 分类1
- 分类2   （ps：分类2会被视作分类1的子分类）
tags：  标签
- 标签1
- 标签2

Markdown常用方法

标题

大标题
========
小标题
--------
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

Markdown提供六级标题

引用

> 引用

结束引用隔一个空行即可

换行

方法1：在结束一段话之后，加两个空格即可
方法2：

字体样式

删除： ~~文本~~
加粗： **文本**
斜体： *文本*
颜色： <span style = "color:color">文本</span>

插入超链接、图片

超链接： [链接名称]（http：//xxxx.com）
插入图  片： 
方法1.![图片名称（可不填）]（图片url）
——在线生成图片url网址（https：//sm.ms）
方法2. ![图片名称（可不填](图片路径)
(~blog/themes/next/source/images/xxx.jpg)

插入代码

方法1：'代码块'
方法2：''' 代码块 '''
方法3：直接缩进四个单位空格/一个tab，再写代码

列表

无序列表
方法1：
     * 列表1
     * 列表2
     * 列表3
方法2：
     - 列表1
     - 列表2
     - 列表3
方法3：
     + 列表1
     + 列表2
     + 列表3
有序列表（自己在前面加数字标识）
     1.列表1
     2.列表2
     3.列表3

分割线

下面写法都可以建立分割线
* * *
***
*****
- - -
---------------------------------------

表格

将表格行与行之间用|隔开
示例
|表格元素1|表格元素2|表格元素3|
|：------|：----：|------：|  
（对齐方式，从左到右依次为左对齐，居中对齐，右对齐）
|   值1  |  值2   |  值3  |

反斜杠\的用处

Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：
\   反斜线
`   反引号
*   星号
_   底线
{}  花括号
[]  方括号
()  括弧
#   井字号
+   加号
-   减号
.   英文句点
!   惊叹号

公式

行内公式  $公式$
陈列公式  $$
            公式
         $$
